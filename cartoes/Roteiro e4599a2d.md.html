Roteiro

Itinerário, descrição de etapas a serem concluídas. Lista de etapas a serem cumpridas para a execução da tarefa.

Para:

- Feedback de progressão;

- Visualizar um resultado antes de completar o processo para alcançá-lo;

- Autonomia para construir um passo a passo para chegar ao mesmo resultado;

- Construção de boas práticas;\

Quando Usar:

- Etapas conhecidas para execução de tarefas;

- Tarefas complexas;

- Participantes com dificuldade de aprendizado;

- Quando é possível analisar as ações tomadas em uma ou mais tarefas;

- Pretende-se construir uma sequência de passos para chegar a um resultado;

Comportamento esperado:

- Os participante executam todos os passos previstos para executar uma tarefa.

Como Usar:

- Roteiro diário e semanal com comparações entre posições;

- Permitir ao participante registrar as ações tomadas durante uma ou mais tarefas;

- Compartilhamento de roteiro para tarefas similares;

Exemplo de uso:

- Um roteiro é acessível na plataforma de desenvolvimento colaborativo, para padronizar o processo para as contribuições dos participantes, desde a escolha da tarefa, alterações, testes, e submissões. Dentro de um roteiro podem haver outros, como um roteiro para os testes de software. O roteiro fica visível durante o trabalho do participante indicando a etapa que está em produção.
