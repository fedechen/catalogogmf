Suporte a Tarefa

Mapa

Representação gráfica para apresentar ou resumir um conjunto de elementos ou passos para execução de uma tarefa.

Para:

- Promover segurança;

- Apoiar decisões;

Quando Usar:

- Deseja-se apresentar a tarefa em relação as demais tarefas;

- Ilustrar uma sequência de passos para se concluir uma tarefa;

- Construção de um roteiro;

- Situar o participante no projeto;

Como Usar:

- Ilustração de passos necessários para se concluir uma tarefa;

- Criar mapas conceituais;

- Expondo partes do mapa conforme os pontos conquistados pelo participante;

- Navegação entre disciplinas;

- Trilha visual com bandeiras a serem conquistadas por um avatar;

Finalidades do Contexto Gamificado (por ordem de relevância):

- Aprendizagem

- Questionário

Característica Open Design:

- Colaboração autônoma, diversa, multidisciplinar, informativa, acessível e dinâmica

Relaciona-se com:

- Cenário/Fantasia

- Pontos

- Progresso

  • Roteiro

Categorias candidatas:

- Atratividade

- Motivação por Realização
